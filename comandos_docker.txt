docker-compose down
docker-compose build --no-cache
docker-compose up -d


Rodar container do quiz app
    docker exec -it quiz_app /bin/bash

Rodar container do db
    docker exec -it quiz_postgres psql -U admin -d questionredis

Containers em execução:
    docker ps

Todos os containers (incluindo parados):
    docker ps -a

Redes ativas:
    docker network ls

Volumes existentes:
    docker volume ls


Se precisar apagar tudo de vez (containers, volumes, redes, imagens), use:
    docker system prune -a --volumes
